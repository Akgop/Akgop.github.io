### 문자열 초기화

---

```python
data = 'Hello World'
data2 = "Hello World"
# 문자열에 따옴표를 넣고 싶다면
data3 = "Hello \"World\""
```

백슬래시는 "이스케이프 문자"로 정해두고 큰따옴표를 출력하는 등 특별한 목적으로 사용한다.

### 문자열 연산

---

```python
a = "Hello"
b = "World"
print(a + " " + b)
>>> Hello World
print(a * 3)       # 반복 출력
>>> HelloHelloHello
```

파이썬의 문자열은 내부적으로 리스트와 동일하게 처리된다.

따라서 문자열 데이터도 인덱싱, 슬라이싱이 가능하다

```python
a = "ABCDEFG"
print(a[2:4])
>>> CD
```
